Fundamentos de HTML

  • Estructura básica de un documento HTML
  • Etiquetas semánticas y su importancia
  • Elementos de texto y enlaces

Módulo 2

Estilos con CSS

  • Selectores y propiedades CSS
  • Modelo de caja y posicionamiento
  • Flexbox y Grid para layouts

Respuestas a las dudas más comunes sobre el curso

No, este curso está diseñado para principiantes absolutos. No necesitas experiencia previa en programación o diseño web. Comenzamos desde lo más básico y avanzamos paso a paso.

Solo necesitas un navegador web moderno (como Chrome o Firefox) y un editor de código como Visual Studio Code (gratuito). Todas las herramientas que utilizaremos son gratuitas y explicamos cómo instalarlas.
