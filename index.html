Contenido del curso

Un programa completo diseñado para avanzar a tu ritmo

Semana 1

Fundamentos de HTML + herramientas

  • Qué es HTML, CSS, navegador, DOM
  • Instalación de VS Code + Live Server
  • Estructura base de un documento HTML + archivo CSS externo
  • Etiquetas: header, main, section, article, footer, etc.
  • Organización del contenido
  • Primer boceto de la landing
  • Selectores básicos (elemento, .clase, #id)
  • Propiedades comunes: color, texto, fondo, padding, margin
  • Sintaxis correcta y buenas prácticas
Semana 2

Maquetación + estilos básicos

  • Maquetado completo de la landing en HTML
  • Imágenes, íconos, contenido de ejemplo
  • Google Fonts
  • Colores, fondo, jerarquía visual
  • Diseño minimalista y moderno
  • Estilizar nav, enlaces, botones
  • Efectos hover y transición básica
  • Estilizar secciones con contraste visual
Semana 3

Layout y responsividad

  • Introducción a Flexbox (contenedor, eje principal/cruzado)
  • Layout de la landing con Flexbox
  • Viewport, unidades relativas (%, rem, vw)
  • Media queries
  • Adaptar la landing a móvil
  • Aplicar todo lo aprendido: estilos finales
  • Detalles visuales y correcciones
Semana 4

Herramientas, repaso y proyecto final

  • git init, add, commit, push
  • Crear repositorio, subir el proyecto
  • Enviar tareas y portafolio por GitHub
  • Revisión de estructura, estilos y responsividad
  • Buenas prácticas y organización del código
  • Preparar el proyecto final
  • Cada alumno muestra su landing
  • Feedback en vivo
  • Orientación para seguir aprendiendo (HTML avanzado, JavaScript, etc.)
